# Assets

Use this folder for:

- Screenshots and images
- PDFs (requirements, specs, exports)
- Any other files you want to attach to the QA docs

Link to them from your markdown, e.g. `![screenshot](assets/screenshot.png)` or `[spec](assets/spec.pdf)`.
